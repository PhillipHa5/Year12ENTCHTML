Welcome To The MFHS Library System

Welcome to the MFHS library system, this system's aim is to act as a replacement for the old library system, and aid in the creation of formal documents and reports. This system allows the user to enter records in the Add Records page, and view and search for records in the Main Table Page. This data is reflected in the graphs located on the View Graphs Page. If you require any help regarding the system, consult the Help Page that contains a guide to how to operate the system.
